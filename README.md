# PetroStack-Casanare.Carlos-Diaz
Se simulo un sistema en Java que gestiona eventos de inspección en pozos petroleros utilizando una estructura tipo pila (LIFO) donde nos da tatos de regristro.
Justificación
Se utiliza una pila porque el último evento registrado es el primero que debe atenderse, lo cual sigue el principio LIFO.
Funcionalidades
* Registrar eventos
* Ver último evento
* Retirar evento
* Deshacer acciones
* Listar eventos
* Validación de datos
Ejecución
bash
javac *.java
java MenuConsola

Uso de IA
Se utilizó ChatGPT como apoyo para mejorar la lógica validaciones y estructura del código verificar las lineas de codigo y buscar soluciones.
